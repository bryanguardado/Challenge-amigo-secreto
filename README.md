# Challenge Amigo Secreto

Un proyecto web interactivo para sortear amigos secretos de manera aleatoria.

## Descripción

Esta aplicación permite a los usuarios:
- Agregar nombres de amigos a una lista
- Visualizar la lista de amigos agregados
- Sortear de manera aleatoria un amigo secreto

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Características

- Interfaz moderna y responsive
- Validación de entradas
- Diseño intuitivo con iconos
- Colores y tipografías atractivas

## Cómo usar

1. Escribe el nombre de un amigo en el campo de texto
2. Haz clic en "Añadir" para agregarlo a la lista
3. Repite el proceso para agregar más amigos
4. Haz clic en "Sortear amigo" para obtener el resultado aleatorio

## Demo

Abre el archivo `index.html` en tu navegador para probar la aplicación.
